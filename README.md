# Re Road
 여행 정보 공유를 위한 웹사이트


여행을 시작하기에 앞서 많은 정보들이 필요하다. 그 중 날씨와 교통정보는 정말 중요하다. 기차와 버스 정보, 날씨 정보를 하나의 웹 사이트에서 볼 수 있도록 하였고 사용자들끼리의 소통을 위해 실시간 채팅기능과 쪽지기능을 추가하였다.

또한 여행준비물, 기억해야할 메모 등이 포함된 나만의 여행플래너 작성도 가능하다. 다양한 정보들로 효율적인 여행계획수립을 가능하게 한다.
